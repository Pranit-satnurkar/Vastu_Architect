Phase 3 — GIS Integration
Sun path analysis:
  Tool: SunCalc.js (free, open source)
  Data: latitude/longitude from user location
  Output: which rooms get morning/evening sun
  Vastu connection: validates East/North room placement

Heat signature:
  Tool: OpenWeatherMap API (free tier)
  Data: local temperature, humidity, wind direction
  Output: optimal wall thickness, window placement
  
Air circulation:
  Tool: Math-based CFD simulation (simplified)
  Data: window positions, room volumes
  Output: airflow score per room

Flood/earthquake zones:
  Tool: USGS API (free), Bhunaksha India (free)
  Data: GPS coordinates
  Output: risk score, foundation recommendations
  
Effort: 6-8 weeks per feature
Cost: mostly free APIs

Phase 4 — Crowd Simulation
Tool: Custom agent-based simulation
      Each person = an agent that moves room to room
      
Metrics:
  Bottleneck detection (narrow corridors)
  CO2 buildup simulation per room volume
  Temperature spread based on occupancy
  
Real world use case:
  "Your corridor is 1.2m wide — 
   3 people can't pass simultaneously"
  "Living room with 8 people needs 
   cross ventilation within 20 minutes"

Effort: 8-10 weeks
This is genuinely novel — no tool does this for homes

Phase 5 — Fire Safety
Tool: Custom pathfinding (A* algorithm)
Data: room connections, door positions (you have this!)
Output: 
  Evacuation route overlay on floor plan
  Time to exit per room
  Safe assembly point suggestion
  Dead end detection 

Effort: 2-3 weeks (A* is well documented)
